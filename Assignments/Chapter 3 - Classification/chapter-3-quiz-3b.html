Quiz 3b: Classification Metrics and Tree Models

Instructions

Time: 30-45 minutes Format: In-class, by-hand (no computer)

Answer all questions. Write code by hand as neatly as possible. Partial credit will be given for correct reasoning even if syntax isn’t perfect.

Section A: Conceptual Questions

Question 1 (4 points)

You’re training a decision tree classifier and observe:

  • Training accuracy: 98%
  • Test accuracy: 72%

a) What problem does this indicate?

b) Name two hyperparameters you could adjust to fix this problem.

c) How do Random Forests address this problem?

Question 2 (3 points)

Explain the purpose of each in 1-2 sentences:

a) The ROC curve

b) AUC (Area Under the Curve)

c) The diagonal line on an ROC plot

Question 3 (2 points)

A spam classifier has precision = 0.95 and recall = 0.70.

a) If you adjust the classification threshold from 0.5 to 0.3, what likely happens to precision and recall?

b) Why does this tradeoff occur?

Question 4 (3 points)

For decision trees:

a) What are two advantages of using decision trees for classification?

b) What is one major disadvantage of decision trees compared to Random Forests?

c) What does “feature importance” tell you in a tree-based model?

Section B: Code Writing

For questions 5-10, assume you have already imported:

import pandas as pd
import numpy as np
from sklearn.tree import DecisionTreeClassifier
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import roc_curve, roc_auc_score
import matplotlib.pyplot as plt

Also assume you have X_train, X_test, y_train, y_test already defined.

Question 5 (3 points)

Write code to fit a Decision Tree classifier with:

  • Maximum depth of 5
  • random_state=42

Then calculate and print the training and test accuracy.

Question 6 (4 points)

Write code to fit a Random Forest classifier with:

  • 100 trees
  • Maximum depth of 5
  • random_state=42

Then extract and display the feature importances (assume features are in a list called feature_names).

Question 7 (3 points)

Write code to create a simple bar plot showing feature importances from a fitted Random Forest model called rf_model.

Use the feature names from a list called feature_names.

Question 8 (4 points)

Write code to plot an ROC curve:

  1. Get probability predictions from a fitted logistic regression model (called log_model)
  2. Calculate false positive rate and true positive rate using roc_curve
  3. Plot the ROC curve with the diagonal reference line
  4. Add a legend showing the AUC score

Question 9 (3 points)

You have predictions from two different models stored in pred_model1 and pred_model2, both applied to the same test set y_test.

Write code to compare which model is better by calculating and printing the accuracy for both models.
