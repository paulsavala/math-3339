Course Information

  • Meeting Pattern: 3 days per week, 50 minutes per class
  • In-class quizzes: Last class of each week is a quiz class. Quizzes are by-hand, no computer.

  • Content Sessions: ~2 per week (new material)

Module 1: Exploratory Data Analysis & AI-Assisted Coding (Weeks 1-3)

Learning Objectives

  • Use AI coding assistants effectively with good prompts
  • Manipulate data with Pandas (loading, filtering, cleaning)
  • Calculate and interpret statistical summaries
  • Create effective visualizations with Seaborn
  • Test data analysis code with assert statements

Module 2: Regression Models (Weeks 4-7)

Learning Objectives

  • Understand proper train/validation/test methodology
  • Use cross-validation for reliable model evaluation
  • Understand and check linear regression assumptions
  • Interpret regression coefficients correctly
  • Use residual analysis to diagnose model problems
  • Apply polynomial features for non-linear relationships
  • Recognize and handle multicollinearity
  • Use Ridge, Lasso, and Elastic Net regularization

Module 3: Classification Models (Weeks 8-10)

Learning Objectives

  • Understand supervised vs unsupervised learning paradigms
  • Understand logistic regression and interpret odds ratios
  • Read and interpret confusion matrices
  • Calculate precision, recall, F1 score and choose appropriately
  • Fit and tune decision trees, random forests, SVMs, k-NN
  • Use ROC curves and AUC for model comparison
  • Handle class imbalance with weights and resampling

Module 4: LLMs for Feature Engineering and Data Extraction (Weeks 11-12)

Learning Objectives

  • Understand how LLMs can transform unstructured text into ML-ready features
  • Write effective prompts for information extraction tasks
  • Integrate LLM APIs (OpenAI, Anthropic, Google) into data pipelines
  • Parse and validate JSON responses from LLMs
  • Calculate and compare API costs across different providers
  • Validate LLM-extracted features for quality and consistency
  • Integrate LLM-generated features into traditional ML models
  • Make informed decisions about when to use LLMs vs. traditional approaches

Schedule

Week Day Topic
11 1 Introduction to LLMs for feature extraction; Zero-shot extraction; Writing effective prompts
11 2 API integration (OpenAI, Anthropic, Google); Parameters (temperature, max_tokens); Cost calculation
11 3 Quiz 4a: Prompt engineering and API basics
12 1 JSON parsing and validation; Handling malformed responses; Building DataFrames
12 2 Few-shot prompting; Quality assessment; Integration with ML pipelines
12 3 Quiz 4b: Integration and quality control

Tips for Success

  1. Do the by-hand work first. Part A of homework builds the foundation you need.
  2. Use AI as a partner, not a replacement. Understand what the code does before using it.
  3. Start homework early. Part B requires iteration with AI assistants.
  4. Come to hackathon days prepared. Review the relevant chapter beforehand.
  5. Use your brain. That’s what it’s there for.
